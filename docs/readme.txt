ClickedMe - Module

Version 3.0.0:
==============
This version is a new developed version for zikula 1.3.x.
If you have ClickedMe <= 3.0.0 installed, you should first remove the 
old module to be sure that all db tables are deleted.
After this remove the old module from your file system and replace it
with the new version and install this version in the Extensions Module
of Zikula.
